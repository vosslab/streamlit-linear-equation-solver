# Linear equation solver

Educational Streamlit tool that solves systems of linear equations via Gaussian elimination, showing each row operation with exact fraction arithmetic and explanatory text. Under active development.

## Quick start

```bash
source source_me.sh && python -m streamlit run linear_solver.py
```
